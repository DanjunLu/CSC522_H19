# CSC522_H19
This is a project for CSC522 in NCSU.
Authors:
Chongwen Sun, Danjun Lu, James Lee and Pengfei Xiong

How can you run our codes:
Step 1: To get data set, please join in the competition on Kaggle: Two Sigma: Using News to Predict Stock Movements
Use news analytics to predict stock price performance. Link: https://www.kaggle.com/c/two-sigma-financial-news
Step 2: start a kernel on Kaggle and add competition data in the draft environment.
Step 3: copy codes in main.py to the kernel and run!
